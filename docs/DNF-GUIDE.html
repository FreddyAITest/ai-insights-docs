📦 DNF Package Manager - Complete Guide

Everything you need to install software on AlmaLinux 9.7

What is DNF?

DNF (Dandified YUM) is the native package manager for AlmaLinux/RHEL/Fedora. It's pre-installed, optimized for your system, and provides better security than Homebrew.

🚀 Quick Start

Install Node.js

sudo dnf install nodejs npm
node --version
npm --version

Install Docker

sudo dnf install docker docker-compose
sudo systemctl start docker
sudo systemctl enable docker
sudo usermod -aG docker $USER

Install Development Tools

sudo dnf group install "Development Tools"

📋 Common Commands

  • dnf install <package> - Install a package
  • dnf remove <package> - Remove a package
  • dnf update - Update all packages
  • dnf search <term> - Search for packages
  • dnf info <package> - Show package info
