🚀 Getting Started with Tavily

1. Sign Up

Go to app.tavily.com/sign-up (2 minutes, no credit card)

2. Get API Key

Dashboard → API Keys → Copy key (starts with tvly-)

3. First Search

const { TavilyClient } = require('tavily');
const tavily = new TavilyClient('YOUR_API_KEY');

const results = await tavily.search('Latest AI developments 2026', {
  search_depth: 'advanced',
  include_answer: true
});

console.log(results);
