📔 Notion Integration Setup

Auto-upload research to your Notion workspace

🔑 STEP 1: Create Notion Integration

  1. Go to notion.com/my-integrations
  2. Click "+ New integration"
  3. Name: "AI Research Automation"
  4. Enable: Read, Update, Insert content
  5. Copy Internal Integration Token (starts with secret_)

📄 STEP 2: Create Page in Notion

  1. Open Notion
  2. Create new page: "AI Research Archive"
  3. Click "..." → "Connect to" → Select your integration

🔗 STEP 3: Get Page ID

  1. Open your page in Notion
  2. Click "..." → "Copy link"
  3. Extract ID from URL: https://notion.so/your-workspace/abc123def456
  4. ID is: abc123def456

⚙️ STEP 4: Update .env

NOTION_API_KEY=secret_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
NOTION_PAGE_ID=abc123def456

🧪 STEP 5: Test Connection

cd /root/.openclaw/workspace/research
export $(cat ../.env | grep -v '^#' | xargs)
node test-notion.js

🚀 Run Research with Notion Upload

node research-automation.js "Your research topic"

The script will automatically upload to Notion!

📊 What Gets Uploaded

  • Research topic title
  • Date and summary
  • All findings with sources
  • Metadata (sources count, date, API used)

Need help? Check test-notion.js or the logs! 🚀
