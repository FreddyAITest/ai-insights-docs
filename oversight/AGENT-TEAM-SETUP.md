## рҹҺҜ How It Works

### 1. You Send a Task
```
You: "Create a landing page for AI Insights"
```

### 2. Manager Routes It
```
Manager: "Routing to web-designer..."
```

### 3. Worker Executes
- Spawns as subagent
- Works with checkpoints
- Manager monitors progress

### 4. Manager Reports
```
Manager: "вң… Landing page complete!
         Preview: https://ai-insights.github.io/landing"
```
